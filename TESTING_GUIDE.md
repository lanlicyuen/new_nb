## 🔧 测试步骤

### 启动应用程序
1. 确保依赖已安装: `npm install`
2. 启动开发服务器: `npm run dev`
3. 访问 http://localhost:5178
4. 或从外部设备访问 http://1.1.1.31:5178

### AI工具管理测试
1. 进入系统后，默认在"AI工具管理"页面
